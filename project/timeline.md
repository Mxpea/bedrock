这个想法非常精妙。将时间维度作为画布的**横坐标轴**，把叙事节点从自由空间布局升级为**时间锚定布局**，能彻底解决复杂叙事中的时间线混乱问题。

我们可以在现有的“故事线画布”基础上，增加一个**时间轴模式开关**。以下是详细的功能与交互设计。

---

### 一、核心概念：双模式画布

在画布右上角增加一个切换开关：

| 模式 | 横坐标含义 | 节点定位方式 | 适用场景 |
| :--- | :--- | :--- | :--- |
| **自由模式** | 无意义 | 手动自由拖拽 | 头脑风暴、逻辑关系梳理。 |
| **时间轴模式** | 时间（可非线性） | 节点吸附到时间锚点，X 坐标由时间决定 | 时间线规划、多线叙事对齐。 |

> **切换行为**：首次切换到时间轴模式时，系统提示“将根据现有节点的时间绑定关系自动排列，未绑定时间的节点将暂存于侧边栏”。

### 三、时间锚点系统（核心创新）

#### 1. 时间锚点的创建与管理

| 操作 | 交互描述 |
| :--- | :--- |
| **创建锚点** | 在画布顶部时间轴标尺上双击，弹出输入框：输入时间名称（如“公元 2123 年”“三体纪元第 5 年”“大低谷时期”）。 |
| **拖拽调整** | 锚点在标尺上可**左右拖拽**以调整相对位置（非线性）。拖动时显示虚线连接跟随。 |
| **设置数值** | 右键锚点“设置时间值”，可输入一个**排序权重数值**（如 100、250、380），用于精确控制非线性间隔。 |
| **删除锚点** | 右键锚点“删除”，该锚点上的节点不会删除，而是解除时间绑定，变为“未锚定”状态。 |
| **锚点样式** | 每个锚点可设置颜色标签，与线段颜色联动（如红色锚点对应主线时间）。 |

#### 2. 非线性时间坐标的实现逻辑

时间轴标尺不是均匀刻度，而是基于**锚点的排序权重**进行比例映射。

- 每个锚点有一个 `order` 数值（可手动设置或由拖拽顺序自动计算）。
- 画布 X 坐标 = 基于 `order` 的线性插值，但作者看到的是**非均匀分布的视觉间隔**。
- **示例**：锚点 A（order=0），锚点 B（order=100），锚点 C（order=300）→ B 到 C 的视觉距离是 A 到 B 的三倍。

> 这允许作者自由表达“时间跳跃”——中间可以存在巨大的留白，而不占用画布空间。

#### 3. 多时间轴并存

| 功能 | 交互描述 |
| :--- | :--- |
| **创建多条时间轴** | 点击「➕ 添加时间轴」，弹出命名窗口（如“主线时间”“罗辑个人时间”“闪回时间线”）。 |
| **时间轴切换** | 顶部下拉菜单选择当前**主时间轴**，画布节点的 X 坐标跟随当前选中的时间轴。 |
| **平行视图** | 点击「平行视图」按钮，画布垂直方向**分割为多个泳道**，每个泳道对应一条时间轴。节点只能在其所属时间轴的泳道内移动。 |
| **时间轴关联** | 节点属性面板中，可为节点选择“绑定到哪个时间轴”。一个节点只能绑定一条时间轴（或暂不绑定）。 |
| **时间轴颜色** | 每条时间轴可设置主题色，该轴下的节点和连线默认继承该颜色。 |

### 四、节点与时间的交互

#### 1. 拖拽吸附

| 操作 | 行为 |
| :--- | :--- |
| **拖拽节点到锚点附近** | 节点自动水平吸附到最近的锚点垂直线，松开后即完成绑定。节点 X 坐标锁定为锚点 X 坐标。 |
| **拖拽调整锚点** | 当移动一个已吸附节点的锚点时，节点跟随移动。 |
| **纵轴自由** | 即使吸附了时间，节点仍可**垂直拖拽**，用于在视觉上区分不同人物线或叙事层。 |
| **解除吸附** | 将节点垂直拖离锚点一定距离后，自动解除时间绑定，变为“自由节点”（在时间轴模式下，自由节点自动收纳到画布左侧的“未锚定区域”）。 |

#### 2. 节点上的时间信息展示

| 节点类型 | 时间信息展示 |
| :--- | :--- |
| **章节节点** | 节点底部显示一行小字：「📅 公元 2123 年 · 第 3 章」。 |
| **情节点节点** | 显示绑定锚点名称，若无则显示「⚡ 未锚定」。 |
| **人物节点** | 不强制绑定时间，但可显示“首次出场时间”。 |

#### 3. 批量时间操作

| 功能 | 描述 |
| :--- | :--- |
| **框选吸附** | 框选多个节点后，右键“吸附到 → [选择锚点]”，批量绑定时间。 |
| **时间偏移** | 选中一个锚点及其所有吸附节点，整体拖拽，可批量调整整段剧情的时间位置。 |
| **时间线对齐** | 提供“按时间排序重新排列纵坐标”功能，让同一时间的节点在垂直方向上自动等距分布。 |

### 五、多时间轴并存的平行视图（泳道模式）

当开启「平行视图」时，画布变为泳道布局：

```
─────────────────────────────────────────────────────────────────────────────
|  时间标尺：  公元2123        公元2125        公元2127        公元2130       |
─────────────────────────────────────────────────────────────────────────────
|  主线时间 █   [Ch1]──────────→[Ch2]──────────→[Ch3]                        |
|  (红色)   █                                                                 |
────────────█────────────────────────────────────────────────────────────────
|  罗辑线   █        [罗辑出场]────→[罗辑领悟]                                |
|  (蓝色)   █                                                                 |
────────────█────────────────────────────────────────────────────────────────
|  闪回线   █   [童年回忆]                                                    |
|  (绿色)   █        └────────────→[青年事件]                                 |
────────────█────────────────────────────────────────────────────────────────
```

**泳道交互**：
- 每条泳道左侧显示时间轴名称和颜色标识。
- 节点只能在**所属泳道**内水平移动（X 轴受锚点吸附），不可跨泳道。
- 节点之间的连线**可以跨泳道**，表示不同时间线之间的叙事关联（如闪回触发）。
- 泳道可上下拖拽排序、可折叠隐藏。

### 六、与正文及数据库的联动

| 联动点 | 功能描述 |
| :--- | :--- |
| **章节内时间标记** | 在正文编辑器中，作者可插入时间戳标记（如 `{@时间:公元2123}`），渲染后在章节阅读页顶部显示该章时间。 |
| **时间轴生成章节索引** | 在阅读端，可根据时间轴数据自动生成“按时间排序的章节索引”，读者可按时间顺序阅读（而非章节发布顺序）。 |
| **人物时间线自动生成** | 当人物节点与章节节点连线（表示出场），且章节绑定了时间锚点，系统可自动为该人物生成一条“出场时间线”，在人物详情页以微型时间轴展示。 |
| **导出时间线图** | 可将当前时间轴视图导出为 SVG/PNG，或导出为 CSV 表格（时间锚点、节点标题、章节链接）。 |

### 十、可能遇到的挑战与应对

| 挑战 | 应对方案 |
| :--- | :--- |
| 非线性标尺的视觉映射算法 | 初期使用固定公式：锚点 X = (order - minOrder) / (maxOrder - minOrder) * 画布宽度 * 0.8 + 边距。 |
| 节点过多时标尺拥挤 | 提供“缩放时间轴”滑块，仅显示主要锚点，次要锚点折叠为小点。 |
| 多时间轴数据同步复杂度 | 每次切换主时间轴，仅重新计算节点的 X 坐标，其他属性保留。 |
| 自由模式与时间轴模式数据兼容 | 切换时不丢失自由布局数据，作为“备用布局”存储，切回时恢复。 |

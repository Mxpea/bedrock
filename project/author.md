个人主页是作者面向读者的**品牌橱窗**，也是外部访客了解创作者的第一触点。结合此前确定的“作者主页完全自定义”需求，以下设计遵循**模块化积木搭建 + 可视化装修**的思路，同时确保未登录访客和内部作者看到的内容有明确边界。

---

### 一、个人主页的两种视图模式

| 视图 | 访问者 | 看到的内容 |
| :--- | :--- | :--- |
| **公开视图** | 未登录访客、普通读者 | 作者公开的个人资料 + **仅可见性为“公开发布”的工作区** |
| **编辑视图** | 主页所有者本人登录后 | 公开视图内容 + **页面右上角悬浮「装修」按钮**，点击进入自定义模式 |

---

### 二、公开视图页面结构（从上到下）

#### 1. 顶部头图区

- **背景头图**：全屏宽度，高度 200-300px（作者可自定义图片或纯色渐变）。
- **头像**：位于头图左下角或居中偏下，圆形，尺寸 100x100px，带白色边框，可上传 GIF 动图。
- **作者名称 + 认证标识**（如有）。
- **一句话简介**：位于头像右侧或下方，短文本。
- **社交链接图标**（可选）：微博、GitHub、个人博客等，作者可在装修时添加。

#### 2. 数据展示条（轻量、无阅读数据焦虑）

仅显示**创作维度**的累计数据：
- 📦 公开工作区数量
- 📝 累计公开章节数
- 📅 加入平台天数

> **注意**：绝不显示总点击量、阅读人数等可能引发作者焦虑的外部数据。

#### 3. 自定义模块区域（核心可拖拽区域）

这是作者自由搭建的部分，系统预设以下模块类型，作者可拖拽排序、增删、配置。

| 模块类型 | 模块内容 | 可配置项 |
| :--- | :--- | :--- |
| **作品陈列柜** | 展示公开工作区的卡片列表。 | 展示样式：瀑布流封面 / 复古书架 / 极简列表；排序方式：手动置顶 / 最近更新 / 自定义顺序；显示数量。 |
| **作者简介** | 富文本自述区域。 | 支持 Markdown 排版，可插入图片、引用。 |
| **创作时间轴** | 按时间倒序显示工作区的更新动态（如“更新了《三体》第23章”）。 | 是否显示具体章节名；时间范围。 |
| **精选数据库展示** | 作者可选择公开某工作区下的若干人物/世界观条目，作为设定集展示。 | 选择工作区 → 勾选要展示的条目 → 展示样式（卡片或列表）。 |
| **留言板** | 读者可在此留言，作者可回复。 | 是否开启审核；是否仅限登录用户留言。 |
| **友情链接** | 作者推荐的其他作者主页或外部链接。 | 链接名称、URL、图标。 |
| **自定义文本/HTML块** | 仅限拥有 Lv.3 权限的作者可添加，运行在沙箱中。 | 输入 HTML/JS/CSS，预览后发布。 |

#### 4. 页脚

- 平台统一页脚（版权信息、举报入口、帮助中心），不可由作者自定义，保证合规。

---

### 三、编辑视图（可视化装修模式）

当作者进入自己的主页并点击「装修」按钮后，页面切换为**搭建模式**。

#### 1. 界面布局

```
[顶部栏]  ← 返回预览  |  保存  |  发布更新  |  撤销  |  重做
-------------------------------------------------------------
|         实时预览画布（显示当前装修效果）                    |
|         （可点击画布中的模块进行选中）                      |
-------------------------------------------------------------
|  左侧组件库        |  右侧属性面板（显示当前选中模块的配置） |
|  - 基础模块        |  - 模块标题                           |
|  - 作品陈列柜      |  - 展示样式： [瀑布流] [书架] [列表]  |
|  - 作者简介        |  - 排序方式： [手动置顶▼]             |
|  - 创作时间轴      |  - 显示数量： [6]                      |
|  - 精选数据库      |  - 选择工作区： [三体 ▼]              |
|  - 留言板          |  - 勾选展示条目： [ ] 罗辑 [√] 叶文洁 |
|  - 友情链接        |                                       |
|  - (Lv.3)自定义块  |                                       |
|                    |  [删除模块]                           |
-------------------------------------------------------------
```

#### 2. 交互细节

- **拖拽排序**：在预览画布中直接上下拖拽模块调整顺序。
- **添加模块**：从左侧组件库点击或拖拽模块到画布中。
- **删除模块**：选中模块后，右侧属性面板底部有删除按钮。
- **实时预览**：修改任何配置项，画布实时更新。
- **保存机制**：点击「保存」仅保存装修草稿，点击「发布更新」才将修改推送到公开视图。

#### 3. 全局样式设置（顶部栏按钮或独立入口）

除模块级配置外，还提供全局样式设置：

- **主题色**：取色器选择，应用于按钮、链接、边框等。
- **背景头图**：上传 / 纯色 / 渐变色。
- **字体方案**：正文、标题的预设字体组合（如“宋体+黑体”“霞鹜文楷+思源”）。
- **模块间距**：紧凑 / 舒适 / 宽松。

---

### 四、访客视角下的交互微细节

- **作品陈列柜卡片**：
    - 鼠标悬停显示工作区简介、最近更新章节。
    - 点击卡片进入该工作区的公开阅读页（仅展示已发布章节）。
- **精选数据库展示**：
    - 点击人物/设定卡片，弹出**快速预览抽屉**（不离开主页），显示该条目的基本信息，并包含“跳转至作品阅读”按钮。
- **留言板**：
    - 简洁的列表样式，支持分页加载。
    - 作者回复直接嵌套在对应留言下方。

---

### 五、后台管理视角（管理员权限）

管理员在后台用户管理列表中，可查看任一作者的主页，并拥有以下操作权限：

- **强制隐藏某模块**（如发现违规内容）。
- **重置该作者主页装修为系统默认模板**。
- **封禁主页访问**（重定向至404或提示“主页已关闭”）。

---

### 六、个人主页 URL 与 SEO 建议

- **URL 格式**：`/author/{作者唯一标识}`（如 `/author/qingya`），作者可在设置中自定义标识。
- **SEO 信息**：页面 Title 自动生成为「作者名 - 平台名」，Description 抓取作者一句话简介。
- **社交分享**：支持分享时展示头图、作者名和精选作品缩略图。

---

### 七、对现有工作区数据的联动

- 当作者在工作区内将某个工作区**从私密改为公开**时，系统提示：“该工作区将在你的个人主页自动上架，是否同步更新主页的作品陈列柜？”（可勾选“不再提示”）。
- 当作者删除了某个已被精选数据库模块引用的条目时，个人主页该模块中对应卡片自动隐藏，并在装修后台显示“已失效”标记。

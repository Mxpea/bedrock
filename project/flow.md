### 一、模块定位与核心价值

| 维度 | 说明 |
| :--- | :--- |
| **名称** | 故事线工作台（Storyline Canvas） |
| **入口** | 工作区左侧导航「🗺️ 大纲」→ 默认进入节点画布视图（可切换回传统树状大纲） |
| **核心目标** | 让作者用**节点+连线**的方式规划故事结构、梳理多线叙事、可视化情节因果链。 |
| **与正文的关系** | 节点可与章节**双向绑定**，点击跳转编辑；章节更新后节点状态自动同步。 |

---

### 二、画布基础交互（仿 ComfyUI 体验）

| 交互 | 描述 |
| :--- | :--- |
| **画布** | 无限延伸的网格背景，支持鼠标中键拖拽平移、滚轮缩放。 |
| **节点** | 矩形卡片，可自由拖拽定位，支持框选多个节点批量移动/删除。 |
| **连线** | 从节点左侧（输入）或右侧（输出）拖出连线到另一节点，形成有向连接。 |
| **右键菜单** | 在画布空白处右键：新建节点、粘贴节点、整理布局、导出截图。在节点上右键：编辑、复制、删除、更改颜色标签、跳转章节。 |
| **快捷键** | Delete 删除选中项、Ctrl+C/V 复制粘贴、Ctrl+Z/Y 撤销重做、Space+拖拽 平移。 |

### 三、节点类型与属性设计

每个节点代表一个叙事单元，类型决定其外观与行为。

| 节点类型 | 图标 | 用途 | 特有属性 |
| :--- | :--- | :--- | :--- |
| **章节节点** | 📖 | 对应一个已存在的正文章节。 | 章节标题、字数、状态（草稿/已发布）、点击跳转编辑。 |
| **情节点节点** | 💡 | 未绑定章节的剧情构思，可后续转为章节节点。 | 标题、描述（Markdown 简写）、标签。 |
| **人物节点** | 👤 | 关联工作区人物库中的角色，用于标记人物出场线。 | 人物名、头像缩略图、点击跳转人物详情。 |
| **分幕/分组节点** | 📁 | 用于包裹一组节点，形成幕、卷、篇章分组。 | 组名称、背景色、可折叠。 |
| **注释节点** | 🗒️ | 自由文本标注，不参与连线逻辑。 | 纯文本内容，可调整尺寸。 |

**节点公共属性**：
- 节点标题（可编辑）
- 自定义颜色标签（用于快速视觉区分）
- 是否折叠（折叠后仅显示标题栏，节省画布空间）

### 四、线段自定义（核心亮点：多线叙事可视化）

连线代表故事流向或逻辑关系。作者可自定义线段样式以区分不同叙事线。

| 线段属性 | 可选项 | 用途举例 |
| :--- | :--- | :--- |
| **颜色** | 调色板预设 12 色 + 取色器 | 红色=主线，蓝色=角色A个人线，绿色=角色B个人线，黄色=暗线/伏笔。 |
| **线型** | 实线 / 虚线 / 点线 | 实线=已写章节，虚线=计划中，点线=隐含关联。 |
| **粗细** | 1px / 2px / 3px | 强调重要故事流。 |
| **箭头** | 有向（默认）/ 无向 / 双向 | 表示因果、时间顺序、相互影响。 |
| **标签** | 在连线上添加短文本 | 如“导致”“启发”“同时发生”。 |

**线段管理**：
- 点击连线选中，右侧弹出属性面板可修改颜色、线型、标签。
- 支持**线段预设**：作者可将当前配色方案保存为“故事线主题”（如“三线叙事-红蓝绿”），一键应用到其他连线。
- 画布右上角显示**图例**：列出当前使用的所有线段颜色及其含义（可手动编辑图例文字）。

### 五、分组功能（幕、卷、篇章）

分组节点是一种特殊的容器节点，用于逻辑收纳。

| 功能 | 交互描述 |
| :--- | :--- |
| **创建分组** | 从左侧工具栏拖出“分组节点”，或框选多个节点后右键“从选中项创建分组”。 |
| **收纳节点** | 将节点拖入分组区域即自动成为组成员，随分组移动。 |
| **折叠分组** | 双击分组标题栏折叠，组内节点隐藏，连线收缩到组边界；再次双击展开。 |
| **分组样式** | 可设置背景色、边框色、标题文字。 |
| **嵌套分组** | 支持分组内再创建子分组（如“第一卷→第一章→第一幕”）。 |

### 六、与正文章节的双向绑定

| 操作 | 行为 |
| :--- | :--- |
| **从章节创建节点** | 在正文章节列表中，右键章节“添加到大纲画布”，自动在画布中心生成一个章节节点。 |
| **节点绑定章节** | 双击情节点节点，在属性面板中点击“关联章节”，可从下拉列表选择已有章节，或点击“新建章节”快速创建空白章节并绑定。 |
| **点击节点跳转** | 点击章节节点或情节点节点（已绑定章节）上的“📖”图标，新标签页打开该章节的编辑器。 |
| **状态同步** | 当正文章节被删除，画布中对应节点变为红色警告态，提示“章节已丢失”。作者可解绑或重新关联。 |
| **进度标记** | 章节节点右下角显示状态标记：🟢 已发布 / 🟡 草稿 / ⚪ 空白（已绑定但无内容）。 |

### 七、画布辅助功能

| 功能 | 描述 |
| :--- | :--- |
| **小地图** | 右下角显示画布缩略图，可拖拽方框快速定位。 |
| **搜索/过滤** | 顶部搜索框，输入关键词高亮匹配节点；可按节点类型、颜色标签过滤显示。 |
| **自动布局** | 提供“整理选中节点”按钮，调用力导向图或树形布局算法自动排列。 |
| **导出/导入** | 导出为 JSON（保留所有节点、连线、样式）；导出为 PNG/SVG 用于分享；支持导入备份。 |
| **版本历史** | 画布自动保存快照，作者可回退到之前的大纲版本。 |
| **只读模式** | 在预览或分享时，可生成只读链接，访客仅可查看画布，无法编辑。 |

### 八、与工作区其他模块的联动

| 联动对象 | 交互方式 |
| :--- | :--- |
| **人物库** | 从右侧“人物库”面板拖拽人物到画布，自动生成人物节点；人物节点可连线到章节节点，表示“该人物在本章出场”。 |
| **世界观库** | 拖拽世界观条目生成设定节点，可连线到章节节点表示“本章揭示该设定”。 |
| **时间轴视图** | （可选）提供一个按时间排序的线性视图，将节点按章节顺序排列，作为画布的补充视图。 |

### 十、与 ComfyUI 的异同及技术可行性

| 对比项 | ComfyUI | 我们的故事线画布 |
| :--- | :--- | :--- |
| 节点用途 | AI 模型工作流 | 故事叙事结构 |
| 连线含义 | 数据流向 | 情节流向 / 逻辑关系 |
| 自定义性 | 插件生态丰富 | 预设类型 + 颜色/线型自定义 |
| 性能要求 | 高（实时渲染） | 中（节点数通常 < 500） |
| 实现复杂度 | 极复杂 | 中等，可基于现有库实现 |

**技术选型建议**：
- 前端画布库：**React Flow** 或 **Vue Flow**（支持节点、连线、分组、自定义样式，文档完善）。
- 自动布局：**Dagre** 或 **ELK**。
- 数据同步：WebSocket 或 自动保存防抖。

### 十一、实现挑战与应对

| 挑战 | 应对方案 |
| :--- | :--- |
| 画布节点过多时性能下降 | 使用虚拟化渲染（React Flow 内置支持），仅渲染可视区域节点。 |
| 线段样式自定义的 UI 复杂度 | 初期仅提供颜色和线型选择，标签功能可二期实现。 |
| 与现有树状大纲的数据兼容 | 两种视图共享底层节点数据，切换视图时双向同步（需约定映射规则）。 |
| 多人协作冲突 | 初期不做实时协作，采用“编辑锁”或最后保存覆盖策略。 |

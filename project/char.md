聚焦到**工作区内的人物模块**，将其设计为直观、高效的“卡片墙 + 详情抽屉”模式，既满足快速浏览，又支持深度编辑。

**组件说明**：

| 区域 | 功能 |
| :--- | :--- |
| **顶部工具栏** | 搜索框（按姓名、别名、描述全文检索）、书签筛选按钮（点亮后仅显示星标人物）、新建人物按钮。 |
| **标签筛选栏** | 作者可自定义标签（系统预设“主角/配角/组织”），点击标签过滤卡片。 |
| **卡片网格** | 响应式网格，每行显示 4-6 张卡片，滚动加载。 |
| **卡片内容** | 头像（可上传，默认占位图）、姓名、一句话简介/身份、出场章节统计、**书签星标**（右上角）。 |
| **空卡片位** | 网格末尾固定一个「+ 新建人物」卡片，点击效果同顶部按钮。 |

### 二、人物卡片交互细节

**1. 悬浮态**
- 鼠标悬停卡片，显示**快速操作按钮**：📝 编辑、🗑️ 删除（带确认）、🔗 复制关联链接。

**2. 点击卡片**
- 右侧滑出**详情抽屉**（宽度约占屏幕 40%，不遮挡全部卡片墙，保持上下文）。

**3. 右键菜单**
- 在卡片上右键，弹出菜单：编辑、复制、删除、置顶、更改标签。

**4. 拖拽排序**
- 卡片支持**手动拖拽排序**，排序结果自动保存为“自定义顺序”。筛选/搜索时暂时按原逻辑排序，清除筛选后恢复自定义顺序。

**抽屉头部按钮**：
- **返回箭头**：关闭抽屉（ESC 键亦可）。
- **星标书签**：收藏此人物。
- **编辑/保存**：默认进入阅读模式，点击编辑按钮后各字段变为可编辑状态（使用内联编辑或表单）。
- **关闭按钮**：关闭抽屉。

**字段交互细节**：
- **标签**：支持多选下拉或自由输入新标签（回车创建）。
- **人物关系图谱**：点击「+ 添加关系」弹出搜索框，可从本工作区人物库中选择目标人物，选择关系类型（预设：亲人/友人/恋人/敌对/从属，支持自定义），连线后自动双向生成关系。
- **出场章节**：自动统计正文中使用 `@罗辑` 引用的章节，点击章节名跳转到该章编辑页（新标签页）。作者也可手动关联未引用的章节。

### 四、与正文编辑器的联动

在正文 Markdown 编辑器中，输入 `@` 触发人物搜索面板，选择后插入一个特殊标记，如 `{@人物:罗辑}`。

渲染时：
- 正文内显示为带下划虚线的人物名，鼠标悬停**弹出迷你卡片**，显示头像、简介和出场次数。
- 点击人物名，可在右侧展开**快速查看面板**（轻量版抽屉），无需离开编辑器。

同时，系统自动在后台为该人物增加一条“出场章节”记录。

### 五、数据存储建议

人物模块的核心数据可存储于一张表 `character`，关键字段：

| 字段 | 类型 | 说明 |
| :--- | :--- | :--- |
| id | UUID | 主键 |
| workspace_id | UUID | 所属工作区 |
| name | string | 姓名 |
| aliases | JSON | 别名数组 |
| avatar_url | string | 头像 URL |
| tags | JSON | 标签数组 |
| summary | string | 一句话简介 |
| description | text (Markdown) | 详细描述（外貌性格） |
| notes | text (Markdown) | 自由备注 |
| relationships | JSON | 关系数据，如 `[{target_id, relation_type, mutual}]` |
| chapter_mentions | JSON | 出场章节引用快照（用于快速展示，也可实时查询） |
| is_starred | boolean | 是否星标 |
| sort_order | integer | 手动排序权重 |
| created_at / updated_at | timestamp | 时间戳 |

### 六、可扩展功能（未来迭代）

- **人物模板**：作者可保存常用人物属性结构为模板，新建时一键套用。
- **批量导入**：支持从 CSV 或已有文档中解析人物名批量创建卡片。
- **人物对比视图**：选择两个人物并排比较属性差异。
- **人物时间轴**：结合章节顺序，生成人物出场时间轴视图。
